# Selenium Python Automation Framework

## Description

A production-ready Selenium automation framework built using Python and Pytest. This project is being developed step by step to demonstrate automation framework design, best coding practices, and real-world QA automation concepts. The framework will gradually include features such as Page Object Model (POM), logging, reporting, API automation, Docker, and CI/CD.

## Application

**Application Under Test (AUT):** SauceDemo

SauceDemo is a demo e-commerce web application used for learning and practicing UI automation. It provides features such as login, product listing, cart management, checkout, and logout, making it ideal for building an end-to-end Selenium automation framework.

**Website:** https://www.saucedemo.com/

## Folder Structure

* `tests` – Test scripts
* `pages` – Page Object classes
* `config` – Configuration files
* `locators` – Element locators
* `utilities` – Reusable helper methods
* `logs` – Execution logs
* `reports` – HTML test reports
* `screenshots` – Screenshots captured during failures
* `testdata` – Test data used by the framework
